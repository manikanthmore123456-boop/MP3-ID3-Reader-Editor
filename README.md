# MP3-ID3-Reader-Editor
MP3 Tag Reader & Editor is a command-line utility built in C that allows you to read, analyze, and modify MP3 metadata (ID3 tags).
It supports essential fields such as Title, Artist, Album, Year, Genre, and Comments, giving users precise control over audio file information.
This project is ideal for:
Students learning file handling in C
Developers exploring ID3 tag structure
Anyone building tools for media management or metadata automation
